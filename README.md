# 🟢 GitHub Green Dot Counter Engine (Day 18)

A clean, responsive mini-project built to understand JavaScript Browser Events, DOM Lifecycle, and DOMContentLoaded event handling.

## 🚀 Features
- **Smart DOM Loading:** Uses `DOMContentLoaded` to ensure scripts execute only after the HTML structure is ready.
- **Event Architecture:** Implements optimized click event listeners.
- **Modern Dark UI:** Clean, responsive card layout inspired by GitHub's developer theme.

## 🛠️ Tech Stack
- **HTML5:** Semantic layout structure.
- **CSS3:** Flexbox positioning, smooth micro-interactions (`hover`, `active` scales), and dark theme colors.
- **JavaScript (ES6):** State management (`count` tracking) and dynamic DOM manipulation (`innerText`).

## 🧠 Key Learnings (Day 18)
- Avoided script crashing by handling the DOM lifecycle correctly.
- Mastered variable scoping inside asynchronous event blocks.
